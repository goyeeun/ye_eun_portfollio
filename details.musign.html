Topic

뮤자인은 디지털플랫폼, 아커머스, 온라인 인프라 구축 등을 제작하고 사용자들에게 배포하며
트렌드에 걸맞는 디자인을 구성하는 기업입니다.

02
Styleguide

point color

검정계열 컬러들을주요 색깔들로 잡으면서 텍스트에 컬러를 중심으로 맞추어 사용자들이 보기에
편안하고 가독성이 잘 나타날 수 있도록 선정하였습니다.

  • #333
  • #888
  • #E7E7E7
  • #FFF

font

  • Noto Sans KR
  • handel gothic
  • Dancing Script

Grid

폰트는 총 3개의 폰트를 설정하였으며 메인 폰트로는 Noto Sans KR 로 사용하고
handel gothic 과 Dancing Script 는 타이틀과 비주얼에 폰트에 사용하여 디자인 요소를
가미하였습니다. 전체적인 사이즈는 1920px로 설정하고 이너사이즈는 1700px로 설정하여 빈여백이 발생하지만 이미지와 텍스트를 정렬함을 전체적으로 채워져 보일 수 있도록 제작하였습니다.

Details

메인 비주얼에는 애니메이트 효과를 적용하고 반응형을 고려해 오른쪽 사이드바에 메뉴를
고정시켜 사용자가 손쉽게 메뉴를 찾을 수 있도록 제작되었습니다.

원 도형에 svg애니메이션을 적용하여 스크롤시 도형이 채워지는 효과를 적용하였으며
이미지에도 동일하게 애니메이션을 적용하여 스크롤시 차오르는 느낌으로 제작하였습니다.

사이드바 메뉴 오픈 시에 skew효과를 적용하여 시각적인 재미요소를 적용하였으며
각 메뉴별로 호버 효과를 적용해 메뉴들이 강조 될 수 있도록 제작하였습니다.
